# scovie

> Scovie is a digital bulletin board for organization that can be displayed on a large screen and accessed through mobile devices. It features a user-friendly interface that displays announcements and the schedule. Additionally, it includes an admin page where the organization's administration can easily edit and manage the information displayed on the bulletin board.

## About

This project uses [Feathers](http://feathersjs.com). An open source framework for building APIs and real-time applications.

## Getting Started

1. Make sure you have [NodeJS](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed.
2. Install your dependencies

    ```
    cd path/to/scovie
    npm install
    ```

3. Start your app

    ```
    npm run compile # Compile TypeScript source
    npm run migrate # Run migrations to set up the database
    npm start
    ```

## Testing

Run `npm test` and all your tests in the `test/` directory will be run.
